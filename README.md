# App sobre Países en Angular

Proyecto de Países en Angular

## Link del proyecto

<https://paises-app-skierhy.netlify.app/>

## Instrucciones para el proyecto

Hacer el build del proyecto
`npm run build`

Correr el proyecto
`npm run start`

## Tecnologías utilizadas en este proyecto

- Angular
- bootstrap
